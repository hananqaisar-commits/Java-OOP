## Projects Overview

### 1. BasicOOP
**Description:** This project introduces basic concepts of Object-Oriented Programming using Java.
**Setup Instructions:** Clone the repository and run `BasicOOP.java`.
**Learning Outcomes:** Understanding of classes, objects, and basic OOP concepts.

### 2. CricketTeamManagement
**Description:** A management system for cricket teams, allowing for player registrations and statistics tracking.
**Setup Instructions:** Ensure Java is installed, compile the project, and run `CricketTeam.java`.
**Learning Outcomes:** Learn about data management and class relationships.

### 3. DeliverySystem
**Description:** A comprehensive delivery management system that handles orders and delivery logistics.
**Setup Instructions:** Follow setup guides in the project folder, run `DeliveryManager.java`.
**Learning Outcomes:** Gain experience with real-world applications of OOP.

### 4. MobileStore
**Description:** A mobile store management application that tracks inventory and sales.
**Setup Instructions:** Compile and run `Mobile.java` following the provided guides.
**Learning Outcomes:** Understand product management and inventory systems.

### 5. OOP_Assignments
**Description:** A collection of assignments demonstrating various OOP concepts.
**Setup Instructions:** See individual assignment instructions.
**Learning Outcomes:** Reinforcement of OOP principles.

### 6. StringAnalyzing_Processing
**Description:** An application that analyzes and processes strings for various functionalities.
**Setup Instructions:** Run `StringAnalyzer.java` after setting up the project.
**Learning Outcomes:** Explore string manipulation and algorithms.

## Individual Java Files

### BankAccountTest.java

### Triangle.java

### University_Staff.java
