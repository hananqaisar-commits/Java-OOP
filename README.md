Object-Oriented Programming Projects (Java)

This repository contains my OOP learning exercises and practice projects developed in Java. It focuses on strengthening core concepts such as classes, objects, constructors, encapsulation, inheritance, and polymorphism.

OOP Basics

Small programs to understand classes, objects, constructors, and core OOP principles.

ATM Simulation

Console-based ATM system with PIN verification, balance inquiry, deposit, and withdrawal features.

Person Class

Encapsulation example using private variables and getter/setter methods.

Inheritance Example

Demonstrates parent and child class relationships and method inheritance.

Calculator (Polymorphism)

Illustrates method overloading using the same method name with different parameters.
